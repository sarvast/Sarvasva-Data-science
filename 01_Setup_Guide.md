## 1. Anaconda Kyun?
Python toh waise bhi install ho sakta hai, par Anaconda ek "All-in-One" kit hai. Isme humein alag-alag libraries (NumPy, Pandas, Jupyter) manual install nahi karni padti. Sab bundled aata hai.

## 2. Downloading & Installation

1.  **Download**:
    - Google pe search karo "Anaconda Individual Edition" ya is link pe jao: [Anaconda Website](https://www.anaconda.com/products/individual)
    - Windows Version download karo (64-Bit Graphical Installer).

2.  **Install**:
    - Setup file run karo.
    - "Next" -> "I Agree" -> "Just Me" select karo.
    - Location default rehne do.
    - **IMPORTANT**: Ek option aayega "Add Anaconda to my PATH environment variable".
        - *Recommendation*: Agar tum beginners ho toh isse **Unchecked** (Red color) rehne do. Hum "Anaconda Prompt" use karenge.

3.  **Finish**: Install hone mein 5-10 minute lagenge. Sabar rakhna.

---

## 3. Jupyter Notebook Kya Hai?
Ye ek aisa notebook hai jahan tum Code likh sakte ho, run kar sakte ho, aur saath hi saath Notes bhi bana sakte ho (jaise ye file padh rahe ho).

### Jupyter Start Kaise Karein?

1.  Windows Search mein type karo: **Anaconda Navigator** aur open karo. (Thoda bhaari app hai, time lega).
2.  Navigator mein bohot saare icons dikhenge. **Jupyter Notebook** dhoondo aur **"Launch"** button dabao.
3.  Ye tumhare Chrome/Edge browser mein khulega.
4.  Wahan `AIML` folder tak navigate karo jo desktop pe banaya hai.
5.  Right side mein "New" -> "Python 3" select karo.
6.  Boom! ðŸ’¥ Tumhari pehli notebook ready hai.

## 4. Keyboard Shortcuts (Pro Banne Ke Liye)
Mouse use karna band karo. Shortcuts seekho:

- **Esc**: Command mode mein jaane ke liye (Cell border blue ho jayega).
- **Enter**: Edit mode mein jaane ke liye (Cell border green ho jayega).
- **Shift + Enter**: Cell run karne ke liye.
- **A**: Current cell ke *Upar* (Above) naya cell add karne ke liye.
- **B**: Current cell ke *Niche* (Below) naya cell add karne ke liye.
- **D, D** (Press D twice): Cell delete karne ke liye.
- **M**: Cell ko Markdown (Notes) banana ke liye.
- **Y**: Cell ko Code wapas banane ke liye.
