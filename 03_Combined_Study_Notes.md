## ðŸŸ¢ Part 1: Machine Learning Kya Hai ? (From Microsoft Course)

**Simple Bhaasha mein:**
Traditional Programming mein hum Computer ko `Rules` aur `Data` dete hain, toh wo `Answers` deta hai.
*Example*: `If a > 10: print("Bada hai")`

**Machine Learning** mein hum Computer ko `Data` aur `Answers` dete hain, aur wo khud `Rules` banata hai.
*Example*: Hum 1000 photos dikhate hain (Data) aur batate hain ye "Billi" hai (Answer). Computer khud pattern (Rule) dhoondta hai ki Billi ke kaan kaise hote hain.

### 2. The History (Jo exams mein nahi aayega par pata hona chahiye)
- **1950s**: Alan Turing ne socha "Can machines think?".
- **1990s**: Deep Blue ne Chess champion ko haraya.
- **2010s**: Deep Learning ka dhamaka (GPUs ki wajah se).

## ðŸŸ¢ Part 2: Types of ML (Ratlo isko)

### 1. Supervised Learning (Teacher Wala)
Jaise bachpan mein Teacher batati thi "A for Apple".
Model ko hum **Labels** (Answer) dete hain.
- **Regression**: Number predict karna (e.g., Zomato delivery time kitna hoga?).
- **Classification**: Category predict karna (e.g., Ye email Spam hai ya nahi?).

### 2. Unsupervised Learning (Khud Se Seekhna)
Koi Teacher nahi, koi Label nahi. Bas data phek do, model pattern dhoondega.
- **Clustering**: Groups banana (e.g., Netflix users ko groups mein baatna - "Action lovers", "Romance lovers").

### 3. Reinforcement Learning (Galti karke seekhna)
Jaise cycle chalana seekhte hain. Girte hain, phir sambhalte hain.
Model ko Reward milta hai sahi karne pe, aur Punishment milti hai galti karne pe. (Used in Robots/Games).

## ðŸŸ¢ Part 3: The Workflow (Kaam kaise hota hai?)

Ek ML Engineer ka 80% time **Data** pe jaata hai, Model pe nahi.

1.  **Define Goal**: Karna kya hai? (e.g., Price predict karna hai).
2.  **Collect Data**: Data kahan se aayega? (CSV, Database, Internet).
3.  **Prepare Data (Sabse Zaroori)**:
    - Missing values bharna.
    - Text ko numbers mein badalna (Computer words nahi samajhta).
4.  **Train Model**: Algorithm choose karo aur data khilao.
5.  **Evaluate**: Test karo ki model kitna paani mein hai.
6.  **Deploy**: App ya Website mein lagao.
